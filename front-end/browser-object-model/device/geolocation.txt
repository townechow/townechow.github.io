c:T18e5,<h2>地理定位 Geolocation API</h2>
<p>根据用户的地理位置提供相关地区服务，已经是非常普遍的一项功能，例如本地生活服务类网站、外卖网站等。HTML5 新功能中提供了获取用户位置的能力。</p>

<p>注意：处于安全考虑，部分最新的浏览器的浏览器只允许通过 HTTPS 协议使用 Geolocation API。在 HTTPS 协议下使用 Geolocation API，浏览器会抛出异常。在开发阶段，127.0.0.1 和 localhost 等本地域在两种协议下均可以使用。</p>
<p>Geolocation API 通过 <code>navigator.geolocation</code> 全局对象进行访问。初次访问时，浏览器会询问用户是否允许共享位置，若用户选择允许则程序获得使用 Geolocation API 权限。判断浏览器是否存在 <code>navigator.geolocation</code> 对象来得知。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">if</span> (navigator.<span class="hljs-property">geolocation</span>) {
  <span class="hljs-comment">// 获取地理位置</span>
} <span class="hljs-keyword">else</span> {
  <span class="hljs-title function_">alert</span>(<span class="hljs-string">&#x27;您的浏览器不支持Geolocation!&#x27;</span>);
}
</code></pre>
<p>获取用户的当前位置，可以调用 <code>navigator.geolocation</code> 的 <code>getCurrentPosition</code> 方法。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">if</span> (navigator.<span class="hljs-property">geolocation</span>){

<p>成功获取位置后，会调用回调函数 <code>success</code>。返回参数 <code>position</code> 对象包含获取位置时的时间戳 <code>timestamp</code> 和坐标信息 <code>coords</code>。<code>coords</code> 对象包含了很多有用的位置数据信息，列举几个常用的属性：</p>
<table>
<thead>
<tr>
<th>属性</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td>latitude</td>
<td>坐标纬度</td>
</tr>
<tr>
<td>longitude</td>
<td>坐标经度</td>
</tr>
<tr>
<td>accuracy</td>
<td>坐标精度，单位为米</td>
</tr>
</tbody>
</table>
<p>当获取位置失败时，会调用回调函数 <code>error</code>。返回的参数 <code>positionError</code> 的 <code>message</code> 属性包含了相关的错误信息描述，<code>positionError.code</code> 标识错误的原因，<code>positionError.code</code> 的值有以下几种。</p>
<table>
<thead>
<tr>
<th>属性</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>UNKNOWN_ERROR(0)</code></td>
<td>其他错误</td>
</tr>
<tr>
<td><code>PERMISSION_DENIED(1)</code></td>
<td>用户拒绝分享位置信息</td>
</tr>
<tr>
<td><code>POSITION_UNAVAILABLE(3)</code></td>
<td>获取用户位置信息失败</td>
</tr>
<tr>
<td><code>TIMEOUT(3)</code></td>
<td>获取用户位置信息超时</td>
</tr>
</tbody>
</table>
<p><code>gerCurrentPosition</code> 方法的参数 <code>options</code> 可以设置以下内容。</p>
<table>
<thead>
<tr>
<th>属性</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td>enableHighAccuray</td>
<td>布尔值，是否获取高精度的位置信息，如果开启，可能会增加响应时间，默认值为 <code>false</code></td>
</tr>
<tr>
<td>timeout</td>
<td>定位超时时间，单位毫秒，如到达时间时没有取得用户位置信息，则触发失败回调函数，默认值为 0，表示无限大</td>
</tr>
<tr>
<td>maximumAge</td>
<td>用户位置信息缓存的最大时间（单位为毫秒），默认值为 0</td>
</tr>
</tbody>
</table>
<p>当用户位置变化时，还可以通过 <code>watchPosition</code> 方法监听用户的位置信息，<code>watchPosition</code> 的参数和 <code>getCurrentPosition</code> 相同。函数执行后返回一个唯一标识，可以使用 <code>clearWatch</code> 方法来取消监听。</p>
