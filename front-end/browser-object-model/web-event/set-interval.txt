c:Teb2,<h1>setInterval</h1>
<p><code>window.setInterval()</code>  方法重复调用一个函数或执行一个代码段，在每次调用之间具有固定的时间延迟。</p>
<p><strong>解决的问题</strong>：由于 JavaScript 的单线程特征，定时器提供了一种跳出单线程限制的方法，即让一段代码在一定毫秒之后，再异步执行。</p>
<h2>语法</h2>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-keyword">let</span> intervalID = <span class="hljs-variable language_">window</span>.<span class="hljs-built_in">setInterval</span>(func, delay[, param1, param2, ...]);
<span class="hljs-keyword">let</span> intervalID = <span class="hljs-variable language_">window</span>.<span class="hljs-built_in">setInterval</span>(code, delay);
</code></pre>
<h3>参数</h3>
<table>
<thead>
<tr>
<th style="text-align:left">参数</th>
<th style="text-align:left">描述</th>
</tr>
</thead>
<tbody>
<tr>
<td style="text-align:left">intervalId</td>
<td style="text-align:left">定时器的唯一辨识符，可以作为参数传给 <code>clearInvterval()</code></td>
</tr>
<tr>
<td style="text-align:left">func</td>
<td style="text-align:left">需要重复调用的函数</td>
</tr>
<tr>
<td style="text-align:left">code</td>
<td style="text-align:left">是另一种语法的应用，是指你想重复执行的一段字符串构成的代码（不推荐使用，不推荐的原因和 <code>eval()</code> 一样）</td>
</tr>
<tr>
<td style="text-align:left">delay</td>
<td style="text-align:left">是每次延迟的毫秒数（一秒等于 1000 毫秒），函数的每次调用会在该延迟之后发生。和 <code>setTimeout</code> 一样，实际的延迟时间可能会稍长一点</td>
</tr>
</tbody>
</table>
<p>需要注意的是，IE 不支持第一种语法中向延迟函数传递额外参数的功能。如果你想要在 IE 中达到同样的功能，你必须使用一种兼容代码。</p>
<h3>注意</h3>
<ul>
<li>定时器的时间间隔设为 0，也会有几毫秒的延迟</li>
<li>在使用 <code>setInterval</code> 和 <code>setTimeout</code> 时最好将其返回值赋值给一个变量，以便取消定时器</li>
<li>在使用 <code>Vue</code> 的时候，<code>setTimeout</code> 和 <code>setInterval</code> 的 <code>this</code> 指向时 window 对象，访问不到组件数据以及方法</li>
<li>在使用 <code>Vue</code> 的时候，路由跳转但不会销毁 <code>setInterval</code> ，但是组件已经销毁了，这会导致问题</li>
<li>在执行线程中 <code>setTimeout</code> / <code>setInterval</code> 无法保证准时执行回调函数的</li>
<li><code>setInterval</code> 调用有可能会被废弃以及 <code>setInterval</code> 的连续执行</li>
</ul>
<h2>停止间歇调用</h2>
<p>使用 <code>clearInterval(timeId)</code> 可以清除间歇调用定时器（定时器还在，只是没调用）。</p>
<h2>使用方法</h2>
<h3>基本用法</h3>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-comment">// 间歇调用的函数</span>
<span class="hljs-keyword">const</span> <span class="hljs-title function_">animate</span> = (<span class="hljs-params"></span>) =&gt; {
  <span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-string">&#x27;间隙调用&#x27;</span>);
};

<span class="hljs-comment">// 间歇调用定时器（表示每过500毫秒执行一次animate函数）</span>
<span class="hljs-keyword">const</span> intervalID = <span class="hljs-built_in">setInterval</span>(animate, <span class="hljs-number">500</span>);

<span class="hljs-comment">// 清除调用</span>
<span class="hljs-built_in">clearInterval</span>(intervalID);
</code></pre>
