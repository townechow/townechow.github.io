c:T10c0,<h1>setTimeout</h1>
<p><code>window.setTimeout()</code> 方法用于在指定的毫秒数后调用函数或计算表达式。</p>
<h2>语法</h2>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-keyword">let</span> timeoutId = <span class="hljs-variable language_">window</span>.<span class="hljs-built_in">setTimeout</span>(func [, delay, param1, param2, ...]);
<span class="hljs-keyword">let</span> timeoutId = <span class="hljs-variable language_">window</span>.<span class="hljs-built_in">setTimeout</span>(func [, delay]);
<span class="hljs-keyword">let</span> timeoutId = <span class="hljs-variable language_">window</span>.<span class="hljs-built_in">setTimeout</span>(code [, delay]);
</code></pre>

<h2>停止超时调用</h2>
<p>使用 <code>clearTimeout(timeId)</code> 可以停止间歇调用定时器（定时器还在，只是没调用）。</p>
<h2>示例</h2>
<h3>基本用法</h3>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-comment">// 超时调用函数</span>
<span class="hljs-keyword">const</span> animate = <span class="hljs-keyword">function</span> (<span class="hljs-params"></span>) {
  <span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-string">&#x27;超时调用&#x27;</span>);
};

<span class="hljs-comment">// 超时调用定时器（经过500毫秒后执行一次animate函数）</span>
<span class="hljs-keyword">const</span> timeoutID = <span class="hljs-built_in">setTimeout</span>(animate, <span class="hljs-number">500</span>);

<span class="hljs-comment">// 清除超时调用</span>
<span class="hljs-built_in">clearTimeout</span>(timeoutID);
</code></pre>
