c:T13ff,<h1>Window 对象视图属性</h1>
<p>这些属性可以 hold 住整个浏览器窗体大小。微软则将这些 API 称为 Screenview 接口。</p>
<h2>浏览器宽高</h2>
<p><code>window.outerWidth</code> 和 <code>window.outerHeight</code> 表示整个浏览器的宽高（以像素为单位），包括侧边栏（如果存在）、窗口镶边（Window Chrome）和窗口调正边框，包含调试窗及浏览器边框。</p>
<pre class="hljs"><code class="language-jsx"><span class="hljs-keyword">import</span> <span class="hljs-title class_">React</span> <span class="hljs-keyword">from</span> <span class="hljs-string">&#x27;react&#x27;</span>;
<span class="hljs-keyword">import</span> img <span class="hljs-keyword">from</span> <span class="hljs-string">&#x27;../../assets/window-view-properties/browser-width-and-height.png&#x27;</span>;

<span class="hljs-keyword">export</span> <span class="hljs-title function_">default</span> () =&gt; <span class="language-xml"><span class="hljs-tag">&lt;<span class="hljs-name">img</span> <span class="hljs-attr">alt</span>=<span class="hljs-string">&quot;浏览器宽高&quot;</span> <span class="hljs-attr">src</span>=<span class="hljs-string">{img}</span> <span class="hljs-attr">width</span>=<span class="hljs-string">{720}</span> /&gt;</span></span>;
</code></pre>
<p>使用如下示例代码：</p>
<pre class="hljs"><code class="language-js"><span class="hljs-variable language_">window</span>.<span class="hljs-property">outerHeight</span>;
<span class="hljs-variable language_">window</span>.<span class="hljs-property">outerHeight</span>;
</code></pre>
<p><code>outterWidth</code> 和 <code>outterHeight</code> 属性为只读属性，没有默认值。</p>
<h2>浏览器网页视口宽高</h2>
<p><code>window.innerWidth</code> 和 <code>window.innerHeight</code> 表示浏览器网页视口的宽高，如果存在垂直/水平滚动条，则包括它，不包括调试窗及浏览器边框。</p>
<pre class="hljs"><code class="language-jsx"><span class="hljs-keyword">import</span> <span class="hljs-title class_">React</span> <span class="hljs-keyword">from</span> <span class="hljs-string">&#x27;react&#x27;</span>;
<span class="hljs-keyword">import</span> img <span class="hljs-keyword">from</span> <span class="hljs-string">&#x27;../../assets/window-view-properties/browser-viewport-width-and-height.png&#x27;</span>;

<span class="hljs-keyword">export</span> <span class="hljs-title function_">default</span> () =&gt; <span class="language-xml"><span class="hljs-tag">&lt;<span class="hljs-name">img</span> <span class="hljs-attr">alt</span>=<span class="hljs-string">&quot;浏览器网页视口宽高&quot;</span> <span class="hljs-attr">src</span>=<span class="hljs-string">{img}</span> <span class="hljs-attr">width</span>=<span class="hljs-string">{720}</span> /&gt;</span></span>;
</code></pre>
<p>使用如下示例代码：</p>
<pre class="hljs"><code class="language-js"><span class="hljs-variable language_">window</span>.<span class="hljs-property">innerWidth</span>;
<span class="hljs-variable language_">window</span>.<span class="hljs-property">innerHeight</span>;
</code></pre>
<p><code>innerWidth</code> 和 <code>innerHeight</code> 属性为只读属性，没有默认值。</p>
<h2>浏览器距屏幕边距</h2>
<p><code>window.screenLeft</code> 和 <code>window.screenTop</code> 分别表示浏览器网页视口的左上角的 X / Y 坐标，该属性为只读属性。（注：在 Firefox 等浏览器中使用的是 <code>screenX</code> 和 <code>screenY</code> 属性）</p>
<p>使用如下示例代码：</p>
<pre class="hljs"><code class="language-js"><span class="hljs-variable language_">window</span>.<span class="hljs-property">screenLeft</span>;
<span class="hljs-variable language_">window</span>.<span class="hljs-property">screenTop</span>;
</code></pre>
<h2>浏览器网页视口滚动偏移</h2>
<p><code>window.pageXOffset</code> 和 <code>window.pageYOffset</code> 表示当前页面相对于网页视口显示区左上角的 X / Y 坐标。</p>
<p>使用如下示例代码：</p>
<pre class="hljs"><code class="language-js"><span class="hljs-variable language_">window</span>.<span class="hljs-property">pageXOffset</span>;
<span class="hljs-variable language_">window</span>.<span class="hljs-property">pageYOffset</span>;
</code></pre>
