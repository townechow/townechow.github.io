c:Tc86,<h1>Screen 对象视图属性</h1>
<h2>屏幕分辨率宽高</h2>
<p><code>screen.width</code> 和 <code>screen.height</code> 分别表示当前浏览器的屏幕的宽高，以像素计算。这两个只读属性，一般用来了解设备的分辨率。除非调整显示器的分辨率，否则这两个值可以看作<strong>常量</strong>，不会发生变化。显示器的分辨率与浏览器设置无关，缩放网页并不会改变分辨率。</p>
<p>使用如下示例代码：</p>
<pre class="hljs"><code class="language-js">screen.<span class="hljs-property">width</span>;
screen.<span class="hljs-property">height</span>;

<span class="hljs-comment">// 同样可通过 Window 对象挂在属性获取</span>
<span class="hljs-variable language_">window</span>.<span class="hljs-property">screen</span>.<span class="hljs-property">width</span>
<span class="hljs-variable language_">window</span>.<span class="hljs-property">screen</span>.<span class="hljs-property">height</span>
</code></pre>
<h3>最佳实践</h3>
<p>可以根据屏幕分辨率，将用户导向不同网页。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">if</span> (screen.<span class="hljs-property">width</span> &lt;= <span class="hljs-number">800</span> &amp;&amp; screen.<span class="hljs-property">height</span> &lt;= <span class="hljs-number">600</span>) {
  <span class="hljs-variable language_">window</span>.<span class="hljs-property">location</span>.<span class="hljs-title function_">replace</span>(<span class="hljs-string">&#x27;small.html&#x27;</span>);
} <span class="hljs-keyword">else</span> {
  <span class="hljs-variable language_">window</span>.<span class="hljs-property">location</span>.<span class="hljs-title function_">replace</span>(<span class="hljs-string">&#x27;wide.html&#x27;</span>);
}
</code></pre>
<h2>屏幕可用工作区宽高</h2>
<p><code>screen.availWidth</code> 和 <code>screen.availHeight</code> 分别表示显示浏览器的屏幕的可用宽高，以像素计算。屏幕可用高度<strong>除去浏览器底部任务栏的屏幕高度</strong>。这两个<strong>只读</strong>属性均为屏幕的像素高度减去系统部件高度之后的值。</p>
<p>使用如下示例代码：</p>
<pre class="hljs"><code class="language-js">screen.<span class="hljs-property">availWidth</span>;
screen.<span class="hljs-property">availHeight</span>;

<span class="hljs-comment">// 同样可通过 Window 对象挂在属性获取</span>
<span class="hljs-variable language_">window</span>.<span class="hljs-property">screen</span>.<span class="hljs-property">availWidth</span>
<span class="hljs-variable language_">window</span>.<span class="hljs-property">screen</span>.<span class="hljs-property">availHeight</span>
</code></pre>
