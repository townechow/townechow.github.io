c:Td9f,<h1>WebRTC</h1>
<p>WebRTC（Web Real-Time Communications）是一项实时通讯技术，它允许网络应用或者站点，在不借助中间媒介的情况下，简历浏览器之间点对点（Peer-to-Peer）的连接，实现视频流和（或）音频流或者其他任意数据的传输。WebRTC 包含的这些标准使用户在无需安装任何插件或者第三方的软件的情况下，创建点对点（Peer-to-Peer）的数据分享和电话会议成为可能。</p>
<h2>引言</h2>
<ul>
<li>通信双方需要先通过服务器交换一些信息</li>
<li>完成信息交换后，通信双方将直接进行连接以传输数据</li>
</ul>

<h2>WebRTC 的组成</h2>
<p>WebRTC 由三大块组成：</p>
<ul>
<li>getUserMedia：负责获取用户本地的多媒体数据，如调起摄像头录像等</li>
<li>RTC PeerConnection：负责建立 P2P 连接以及传输多媒体数据</li>
<li>RTC DataChannel：提供一个信令通道，在游戏里面信令是实现互动的重要元素</li>
</ul>
<h2>文件共享</h2>
<p>目前，数据通道支持如下类型：</p>
<ul>
<li>String：JavaScript 基本的字符串</li>
<li>Blob（Binary large object）：二进制大对象</li>
<li>ArrayBuffer：确定数组长度的数据类型</li>
<li>ArrayBufferView：基础的数组视图</li>
</ul>
<p>其中，Blob 类型是一个可以存储二进制文件的容器，结合 HTML5 相关文件读取 API，可以实现文件共享的功能。</p>
