c:T11c0,<h1>全屏 Fullscreen API</h1>
<p>全屏 API 为使用用户的整个屏幕展现网络内容提供了一种简单的方式。这种 API 让你可以简单地控制浏览器，使得一个元素与其子元素，如果存在的话，可以占据整个屏幕，并在此期间，从屏幕上隐藏所有的浏览器用户界面以及其他应用。</p>
<h2>激活全屏模式</h2>
<p>通过 DOM 元素的 <code>requestFullScreen()</code> 方法可以激活全屏模式。</p>
<p>最常见的场景是视频窗口的全屏模式：</p>
<pre class="hljs language-html"  style=--lang:"html" ><code><span class="hljs-tag">&lt;<span class="hljs-name">video</span> <span class="hljs-attr">id</span>=<span class="hljs-string">&quot;_video&quot;</span> <span class="hljs-attr">src</span>=<span class="hljs-string">&quot;blob:https://wwww.example.com/&quot;</span>&gt;</span><span class="hljs-tag">&lt;/<span class="hljs-name">video</span>&gt;</span>
</code></pre>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-keyword">const</span> video = <span class="hljs-variable language_">document</span>.<span class="hljs-title function_">getElementById</span>(<span class="hljs-string">&#x27;_video&#x27;</span>);

video.<span class="hljs-title function_">addEventListener</span>(<span class="hljs-string">&#x27;click&#x27;</span>, <span class="hljs-keyword">function</span>(<span class="hljs-params"></span>) {
  <span class="hljs-keyword">if</span> (video.<span class="hljs-property">requestFullscreen</span>) {

  }

<h3>显示差异</h3>
<p>Gecko 和 Webkit 内核实现效果存在关键差异：</p>
<ul>
<li>Gecko 会自动为元素添加 CSS 规则，使其拉伸以填满屏幕 <code>width: 100%;height: 100%;</code></li>
<li>Webkit 没有这样做，相反地，它将全屏元素剧中，并以同样的大小显示，而屏幕的其他部分为黑色</li>
</ul>
<p>为了在 Webkit 中获得相同的全屏行为，你需要自行元素添加 <code>width: 100%;height: 100%</code></p>
<pre class="hljs language-css"  style=--lang:"css" ><code><span class="hljs-selector-id">#video</span>:-webkit-full-screen {
  <span class="hljs-attribute">width</span>: <span class="hljs-number">100%</span>;
  <span class="hljs-attribute">height</span>: <span class="hljs-number">100%</span>;
}
</code></pre>
<p>另一个方面，如果你尝试在 Gecko 上模拟 Webkit 的行为，你需要把你想要呈现的元素放在另一个实际调整为全屏幕的元素中，并使用 CSS 规则调整内部元素，达到你想要的样式。</p>
<h3>通知</h3>
<p>当成功进入全屏模式时，包含该元素的文档会收到 <code>fullscreenchange</code> 事件。当退出全屏模式时，文档会再次收到该事件。注意此事件本身，不管在文档进入和退出全屏模式时，都不会提供任何信息，但如果文档有一个非空的 <code>fullscreenElement</code>，你就可以得知你处于全屏模式中。</p>
<h2>退出全屏模式</h2>
<p>通过调用 <code>document.exitFullscreen()</code> 方法能退出全屏模式。</p>
<h2>其他信息</h2>
<p><code>document</code> 提供了一些额外的信息，在开发全屏网络应用时能够使用。</p>

</tr>
