c:T1654,<h1>HTMLAudioElement</h1>
<pre class="hljs"><code>EventTarget &lt;- Node &lt;- Element &lt;- HTMLElement &lt;- HTMLMediaElement &lt;- HTMLAudioElment
</code></pre>
<p>HTML5 <code>&lt;audio&gt;</code> 元素用于在文档中嵌入音频内容。<code>&lt;audio&gt;</code> 元素可以包含一个或多个音频资源，这些音频资源可以使用 <code>src</code> 属性或者 <code>&lt;source&gt;</code> 元素来进行描述：浏览器将会选择最合适的一个来使用。也可以使用 MediaStream 将这个元素用于流式媒体。</p>
