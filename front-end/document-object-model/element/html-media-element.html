HTMLMediaElement

HTML 媒体元素接口在属性和方法中添加了 HTML 元素来支持基础的媒体相关的能力，就像 audio 和 video 一样。HTML 视频元素和 HTML 音频元素元素都继承自此接口。

EventTarget <- Node <- Element <- HTMLElement <- HTMLMediaElement

特性

从父级 HTMLElement、Element、Node 和 EventTarget 继承属性。
