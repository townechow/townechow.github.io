c:T1403,<h1>偏函数</h1>
<p>维基百科中对偏函数（Partial）的定义为：</p>
<blockquote>
<p>In computer science, partial application(or partial function application) refers to the process of fixing a number of arguments to a function, producing another function of smaller arity.</p>
</blockquote>
<p>在计算机科学中，局部应用是指固定一个函数的一些参数，然后产生另一个更小元的函数。</p>
<p>什么是元？</p>
<p>元是指函数参数的个数，比如一个带有两个参数的函数被称为二元函数。</p>
<p>🌰 <strong>示例：</strong></p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">function</span> <span class="hljs-title function_">add</span>(<span class="hljs-params">a, b</span>) {
  <span class="hljs-keyword">return</span> a + b;
}

<span class="hljs-comment">// 执行 add 函数，一次传入两个参数即可</span>
<span class="hljs-title function_">add</span>(<span class="hljs-number">1</span>, <span class="hljs-number">2</span>); <span class="hljs-comment">// 3</span>

<span class="hljs-comment">// 假设有一个 partial 函数可以做到局部应用</span>
<span class="hljs-keyword">var</span> addOne = <span class="hljs-title function_">partial</span>(add, <span class="hljs-number">1</span>);

<span class="hljs-title function_">addOne</span>(<span class="hljs-number">2</span>); <span class="hljs-comment">// 3</span>
</code></pre>
<p>偏函数与柯里化十分相像：</p>
<ul>
<li>**柯里化：**将多参数函数转换成多个单参数函数，也就是将一个 n 元函数转换成 n 个一元函数</li>
<li>**偏函数：**则是固定一个函数的一个或多个参数，也就是将一个 n 元函数转换成一个 n - x 元函数</li>
</ul>
<h2>实际应用</h2>
<p><code>bind</code> 函数可以让我们传入一个或多个预设的参数，之后返回一个新函数，并拥有指定的 <code>this</code> 值和预设参数。当绑定函数被调用时，这些参数会被插入到目标函数的参数列表的开始位置，传递给绑定函数的参数跟在它们后面。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">function</span> <span class="hljs-title function_">addition</span>(<span class="hljs-params">x, y</span>) {
  <span class="hljs-keyword">return</span> x + y;
}

<span class="hljs-keyword">const</span> plus5 = addition.<span class="hljs-title function_">bind</span>(<span class="hljs-literal">null</span>, <span class="hljs-number">5</span>);

<span class="hljs-title function_">plus5</span>(<span class="hljs-number">10</span>);
<span class="hljs-comment">// 15</span>

<span class="hljs-title function_">plus5</span>(<span class="hljs-number">25</span>);
<span class="hljs-comment">// 30</span>
</code></pre>
<p>我们预先传入参数 <code>5</code>，并返回一个新函数赋值给 <code>plus5</code>，此函数可以接受剩余的参数。调用 <code>plus5</code> 传入剩余参数 <code>10</code> 得到最终结果 <code>15</code>，又传入参数 <code>20</code> 得到结果 <code>30</code>。偏函数通过设定预设值，帮助哦我们实现代码上的复用。</p>
