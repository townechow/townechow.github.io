c:T1088,<h1>级联函数</h1>
<p><strong>级联函数</strong> 也叫 <strong>链式函数</strong>，是一种在一个对象上使用一条连续的代码来重复调用不同方法的技巧。这种技巧在 jQuery 和其他一些 JavaScript 库中很流行，它甚至也是一些 JavaScript 原生方法的内在特性，比如常见的字符串方法。一定程度上可以减少代码量，提高代码可读性，缺点是它占用了函数的返回值。</p>
<p>级连函数的表达形式如下所示：</p>
<pre class="hljs"><code class="language-js"><span class="hljs-comment">// jQuery</span>
$(<span class="hljs-string">&#x27;#wrapper&#x27;</span>).<span class="hljs-title function_">fadeOut</span>().<span class="hljs-title function_">html</span>(<span class="hljs-string">&#x27;Welcome, Sir&#x27;</span>).<span class="hljs-title function_">fadeIn</span>();

<span class="hljs-comment">// 字符串操作</span>
<span class="hljs-string">&#x27;kankuuii&#x27;</span>.<span class="hljs-title function_">replace</span>(<span class="hljs-string">&#x27;k&#x27;</span>, <span class="hljs-string">&#x27;R&#x27;</span>).<span class="hljs-title function_">toUpperCase</span>().<span class="hljs-title function_">substr</span>(<span class="hljs-number">0</span>, <span class="hljs-number">4</span>);
<span class="hljs-comment">// &#x27;RANK&#x27;</span>
</code></pre>
<h2>实现方法</h2>
<p>要使用级联函数，我们只需要在每个函数中返回 <code>this</code> 对象（也就是后面方法中操作的对象）。操作的对象就会在执行完一个函数后继续调用往后的方法，即实现了链式操作了。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">function</span> <span class="hljs-title function_">Person</span>(<span class="hljs-params"></span>) {
  <span class="hljs-variable language_">this</span>.<span class="hljs-property">name</span> = <span class="hljs-string">&#x27;&#x27;</span>;
  <span class="hljs-variable language_">this</span>.<span class="hljs-property">age</span> = <span class="hljs-number">0</span>;
  <span class="hljs-variable language_">this</span>.<span class="hljs-property">weight</span> = <span class="hljs-number">10</span>;
}

<span class="hljs-title class_">Person</span>.<span class="hljs-property"><span class="hljs-keyword">prototype</span></span> = {
  <span class="hljs-attr">setName</span>:<span class="hljs-keyword">function</span>(<span class="hljs-params">name</span>){
    <span class="hljs-variable language_">this</span>.<span class="hljs-property">name</span> = name;
    <span class="hljs-keyword">return</span> <span class="hljs-variable language_">this</span>;
  },
  <span class="hljs-attr">setAge</span>:<span class="hljs-keyword">function</span>(<span class="hljs-params">age</span>){
    <span class="hljs-variable language_">this</span>.<span class="hljs-property">age</span> = age;
    <span class="hljs-keyword">return</span> <span class="hljs-variable language_">this</span>;
  },
  <span class="hljs-attr">setWeight</span>:<span class="hljs-keyword">function</span>(<span class="hljs-params">weight</span>) {
    <span class="hljs-variable language_">this</span>.<span class="hljs-property">weight</span> = weight;
    <span class="hljs-keyword">return</span> <span class="hljs-variable language_">this</span>;
  }
}

<span class="hljs-keyword">var</span> uzi = <span class="hljs-keyword">new</span> <span class="hljs-title class_">Person</span>();

uzi.<span class="hljs-title function_">setName</span>(<span class="hljs-string">&#x27;Uzi&#x27;</span>).<span class="hljs-title function_">setAge</span>(<span class="hljs-number">22</span>).<span class="hljs-title function_">setWeight</span>(<span class="hljs-number">160</span>);

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(uzi);
<span class="hljs-comment">// { name: &quot;Uzi&quot;, age: 22, weight: 160 }</span>
</code></pre>
<p>通过工厂函数创建实例对象，由于所有对象都会继承其原型对象的属性和方法，所以我们可以让定义原型对象中的那几个方法都返回用以调用方法的实例对象的引用，这样既可以对那些方法进行链式调用。</p>
