c:T8e4,<h1>寄生式继承</h1>
<p><strong>寄生式继承（Parasitic Inheritance）</strong>：创建一个仅用于封装继承过程的函数，在函数内部以某种方式增强对象</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">function</span> <span class="hljs-title function_">creator</span>(<span class="hljs-params">origin</span>) {
  <span class="hljs-comment">// 以 origin 为原型对象创建一个新对象</span>
  <span class="hljs-keyword">let</span> clone = <span class="hljs-title class_">Object</span>.<span class="hljs-title function_">create</span>(origin);

  <span class="hljs-comment">// 以某种方式来增强这个对象</span>
  clone.<span class="hljs-property">sayHi</span> = <span class="hljs-keyword">function</span> (<span class="hljs-params"></span>) {
    <span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-string">&#x27;Hello world!&#x27;</span>);
  };

  <span class="hljs-comment">// 返回这个对象</span>
  <span class="hljs-keyword">return</span> clone;
}

<span class="hljs-keyword">let</span> friendship = {
  <span class="hljs-attr">name</span>: <span class="hljs-string">&#x27;Uzi&#x27;</span>,
  <span class="hljs-attr">friends</span>: [<span class="hljs-string">&#x27;Amy&#x27;</span>, <span class="hljs-string">&#x27;Ben&#x27;</span>, <span class="hljs-string">&#x27;Tom&#x27;</span>],

<p>在主要考虑对象而 <strong>不是自定义类型和构造函数的情况下</strong>，寄生式继承也是一种有用的模式。前面示范继承模式时使用的 <code>Object</code> 函数不是必需的，任何能够返回新对象的函数都适用于此模式。</p>
<blockquote>
<p>⚠️ <strong>注意</strong>： 使用寄生式继承来为对象添加函数，会由于不能做到函数复用而降低效率；这一点与借用构造函数模式类似。</p>
