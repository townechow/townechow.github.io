c:Tef9,<h1>Proxy - revocable</h1>
<p><code>Proxy.revocable()</code> 方法可以用于创建一个可撤销的代理对象。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-title class_">Proxy</span>.<span class="hljs-title function_">revocable</span>(target, handler);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
<th>类型</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>target</code></td>
<td>将用 Proxy 封装的目标对象</td>
<td>any</td>
</tr>
<tr>
<td><code>handler</code></td>
<td>一个对象，其属性是一批可选的函数，这些函数定义了对应的操作被执行时代理的行为</td>
<td>object</td>
</tr>
</tbody>
</table>
<p>返回一个包含了代理对象本身和它的撤销方法的可撤销 Proxy 对象。</p>
<h2>说明</h2>
<p>该方法的返回值是一个对象，其结构为：</p>
<pre class="hljs"><code class="language-js">{
    <span class="hljs-string">&quot;proxy&quot;</span>: proxy,
    <span class="hljs-string">&quot;revoke&quot;</span>: revoke
}
</code></pre>
<p>其中：</p>
<ul>
<li><code>proxy</code>：表示新生成的代理对象本身，和用一般方式 <code>new Proxy(target, handler)</code> 创建的代理对象没什么不同，只是它可以被撤销掉</li>
<li><code>revoke</code>：撤销方法，调用的时候不需要加任何参数，就可以撤销掉和它一起生成的哪个代理对象</li>
</ul>
<p>一旦某个对象被撤销，它将变得几乎完成不可调用，在它身上执行任何的 <strong>可代理操作</strong> 都会抛出 TypeError 异常（注意，可代理操作一共有 14 种，执行这 14 种操作以外的操作不会抛出异常）。一旦被撤销，这个代理对象便不可能被直接恢复到原来的状态，同时</p>
<h2>示例</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">const</span> revocable = <span class="hljs-title class_">Proxy</span>.<span class="hljs-title function_">revocable</span>(
  {},
  {
    <span class="hljs-title function_">get</span>(<span class="hljs-params">target, name</span>) {
      <span class="hljs-keyword">return</span> <span class="hljs-string">&#x27;[[&#x27;</span> + name + <span class="hljs-string">&#x27;]]&#x27;</span>;
    },
  }
);

<span class="hljs-keyword">const</span> proxy = revocable.<span class="hljs-property">proxy</span>;

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(proxy.<span class="hljs-property">foo</span>);
<span class="hljs-comment">// &quot;[[foo]]&quot;</span>

revocable.<span class="hljs-title function_">revoke</span>();

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(proxy.<span class="hljs-property">foo</span>);
<span class="hljs-comment">// Uncaught TypeError: Cannot perform &#x27;get&#x27; on a proxy that has been revoked</span>

proxy.<span class="hljs-property">foo</span> = <span class="hljs-number">1</span>;
<span class="hljs-comment">// Uncaught TypeError: Cannot perform &#x27;set&#x27; on a proxy that has been revoked</span>

<span class="hljs-keyword">delete</span> proxy.<span class="hljs-property">foo</span>;
<span class="hljs-comment">// Uncaught TypeError: Cannot perform &#x27;deleteProperty&#x27; on a proxy that has been revoked</span>

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-keyword">typeof</span> proxy);
<span class="hljs-comment">// &quot;object&quot;</span>
<span class="hljs-comment">// 因为 typeof 不属于可代理操作</span>
</code></pre>
<p><code>Proxy.revocable()</code> 的一个使用场景是，目标对象不允许直接访问，必须通过代理访问，一旦访问结束，就收回代理权，不允许再次访问。</p>
