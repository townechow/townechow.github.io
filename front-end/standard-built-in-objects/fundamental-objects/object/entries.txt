c:Tfb7,<h1>Object.entries</h1>
<p>⭐️ <code>ES2017(ES8)新特性</code></p>
<p><code>Object.entries()</code> 方法用于枚举指定对象并返回以键值对组成的数组为元素的二维数组。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js"><span class="hljs-title class_">Object</span>.<span class="hljs-title function_">entries</span>(obj);
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">ObjectConstructor</span> {
  values&lt;T&gt;(<span class="hljs-attr">o</span>: { [<span class="hljs-attr">s</span>: <span class="hljs-built_in">string</span>]: T } | <span class="hljs-title class_">ArrayLike</span>&lt;T&gt;): T[];

  <span class="hljs-title function_">values</span>(<span class="hljs-attr">o</span>: {}): <span class="hljs-built_in">any</span>[];
}
</code></pre>

<p>返回值：</p>
<p>返回给定对象自身可枚举 Property 的键值对数组。</p>
<h2>方法说明</h2>
<p>给定对象自身可枚举属性的键值对数组，其排列与使用 <a href="/front-end/basic-concept/statements-and-declarations/iteration-statement/the-for-in-statement">for-in</a> 循环遍历该对象时返回的顺序一致，区别在于 <code>for-in</code> 循环也枚举原型链中的属性。</p>
