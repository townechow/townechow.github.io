c:Td5e,<h1>Object.isFrozen</h1>
<p><code>Object.isFrozen()</code> 方法用来检测指定对象是否已被冻结。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-title class_">Object</span>.<span class="hljs-title function_">isFrozen</span>(O);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
<th>类型</th>
</tr>
</thead>
<tbody>
<tr>
<td>O</td>
<td>指定用于检测的对象</td>
<td>object</td>
</tr>
</tbody>
</table>
<p>返回 Boolean 类型的值表示用于检测的对象是否被冻结。</p>
<h2>描述</h2>
<p>被冻结的对象不可扩展，所有 Property 均不可配置，且所有数据属性（即没有 <code>getter</code> 或 <code>setter</code> 组件的访问器的属性 ）都是不可写的。</p>
<h2>示例</h2>
<h3>基本示例</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">let</span> foo = {
  <span class="hljs-attr">a</span>: <span class="hljs-number">1</span>,
  <span class="hljs-attr">b</span>: <span class="hljs-number">2</span>,
};

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-title class_">Object</span>.<span class="hljs-title function_">isFrozen</span>(foo));
<span class="hljs-comment">// false</span>

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-title class_">Object</span>.<span class="hljs-title function_">freeze</span>(foo));
<span class="hljs-comment">// {a: 1, b: 2}</span>

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-title class_">Object</span>.<span class="hljs-title function_">isFrozen</span>(foo));
<span class="hljs-comment">// true</span>

foo.<span class="hljs-property">a</span> = <span class="hljs-number">3</span>;

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(foo);
<span class="hljs-comment">// { a: 1, b: 2 }</span>
</code></pre>
<p><code>Object.freeze</code> 方法实际上会在现有对象上调用 <code>Object.seal()</code> 方法，并把所有现有属性的 <code>writable</code> 描述符置为 <code>false</code>。</p>
