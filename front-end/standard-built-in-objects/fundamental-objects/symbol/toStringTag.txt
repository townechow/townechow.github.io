c:Tb77,<h1>Symbol.toStringTag</h1>
<p>对象的<code>Symbol.toStringTag</code>属性，指向一个方法。在该对象上面调用<code>Object.prototype.toString</code>方法时，如果这个属性存在，它的返回值会出现在<code>toString</code>方法返回的字符串之中，表示对象的类型。也就是说，这个属性可以用来定制<code>[object Object]</code>或<code>[object Array]</code>中<code>object</code>后面的那个字符串。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-comment">// 例一</span>
({ [<span class="hljs-title class_">Symbol</span>.<span class="hljs-property">toStringTag</span>]: <span class="hljs-string">&#x27;Foo&#x27;</span> }.<span class="hljs-title function_">toString</span>());
<span class="hljs-comment">// &quot;[object Foo]&quot;</span>

<span class="hljs-comment">// 例二</span>
<span class="hljs-keyword">class</span> <span class="hljs-title class_">Collection</span> {

}
