c:T9f4,<h1>encodeURIComponent</h1>
<p><code>encodeURIComponent()</code> 函数用于对统一资源标识符（URI）的有效组件进行编码，并返回编码后的字符串。</p>
<p>该函数属于 <code>Global</code> 对象，所有主流浏览器均支持该函数。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-built_in">encodeURIComponent</span>(<span class="hljs-title class_">URIString</span>);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>类型</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>URIString</code></td>
<td><code>String</code> 类型</td>
<td>需要编码的 URI 组件字符串</td>
</tr>
</tbody>
</table>
<p><code>encodeURIComponent()</code> 函数的返回值是 <code>string</code> 类型，返回一个编码后的 URI 组件字符串。</p>
<h2>说明</h2>
<ul>
<li>如果要对使用<code>encodeURIComponent()</code> 函数编码后的 URI 组件字符串进行解码，请使用 <a href="/front-end/standard-built-in-objects/the-global-object/function-properties/decodeURIComponent">decodeURIComponent()</a> 函数。</li>
<li><code>encodeURIComponent()</code> 函数会编码所有的字符。如果你想把 URI 当作请求参数传递，那么你可以使用本函数。如果你只是想编码一个带有特殊字符（比如中文）的 URI，这个 URI 用作请求地址，请使用 <a href="/front-end/standard-built-in-objects/the-global-object/function-properties/encodeURI">encodeURI</a> 函数。</li>
</ul>

<h3>代码示例</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-comment">// 原 URI 组件</span>
<span class="hljs-keyword">var</span> origin = <span class="hljs-string">&#x27;ftp://192.168.0.100/共享文件夹&#x27;</span>;

<span class="hljs-comment">// 编码 URI 组件</span>

<span class="hljs-comment">// 解码 URI 组件</span>

<span class="hljs-variable language_">document</span>.<span class="hljs-title function_">writeln</span>(decodedUri); <span class="hljs-comment">// ftp://192.168.0.100/共享文件夹</span>
</code></pre>
