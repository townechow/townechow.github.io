c:Ta39,<h1>encodeURI</h1>
<p><code>encodeURI()</code> 函数可把 URI 字符串采用 UTF-8 编码格式转化成 escape 格式的字符串。</p>
<p>该函数属于 <code>Global</code> 对象，所有主流浏览器均支持该函数。</p>
<h2>语法</h2>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-built_in">encodeURI</span>(<span class="hljs-title class_">URIString</span>);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>类型</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>URIString</code></td>
<td><code>String</code> 类型</td>
<td>需要编码的 URI 字符串</td>
</tr>
</tbody>
</table>
<p>该方法返回一个已经编码的 URI 字符串。</p>
<h2>说明</h2>
<p>如果要对使用 <code>encodeURI()</code> 函数编码的 URI 字符串进行解码，请使用 <code>decodeURI()</code> 函数。</p>
<p><code>encodeURI()</code> 函数不编码字符有 82 个 <code>!</code>、<code>#</code>、<code>$</code>、<code>'</code>、<code>(</code>、<code>)</code>、<code>*</code>、<code>+</code>、<code>,</code>、<code>-</code>、<code>.</code>、<code>/</code>、<code>:</code>、<code>;</code>、<code>=</code>、<code>?</code>、<code>@</code>、<code>_</code>、<code>~</code>、<code>0-9</code>、<code>a-z</code>、<code>A-Z</code> 。</p>
<p>如果你只是想编码一个带有特殊字符（比如中文）的 URI，这个 URI 用作请求地址，请使用本函数。</p>
<p>如果你想把 URI 当作请求参数传递，那么你可以使用 <code>encodeURIComponent()</code> 函数。<code>encodeURIComponent()</code> 函数会编码所有的字符。</p>

<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-comment">// 原URI</span>

</code></pre>
