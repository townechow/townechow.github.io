c:Tab3,<h1>isFinite</h1>
<p><code>isFinite()</code> 函数用于判断指定数字是否是有限值。如果指定的数字为 <code>NaN</code>、<code>Infinity</code>、<code>-Infinity</code>，则返回 <code>false</code>，其他数字均返回 <code>true</code>。</p>
<p>该函数属于 <code>Global</code> 对象，所有主流浏览器均支持该函数。</p>
<h2>语法</h2>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-built_in">isFinite</span>(number);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>类型</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>number</code></td>
<td><code>Number</code> 类型</td>
<td>需要检测的值</td>
</tr>
</tbody>
</table>
<p>如果参数<code>number</code> 不是 Number 类型（如字符串、函数等），也返回 <code>false</code>。</p>
<p><code>isFinite()</code> 函数的返回值是 Boolean 类型。</p>
<ul>
<li>当指定的数字为 <code>NaN</code>、正无穷、负无穷时，返回 <code>false</code>；</li>
<li>除上述三种 Number 类型外的数字均返回 <code>true</code>。</li>
</ul>
<h2>示例</h2>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-comment">// false situation</span>
<span class="hljs-built_in">isFinite</span>(<span class="hljs-title class_">NaN</span>);
<span class="hljs-comment">// false</span>
<span class="hljs-built_in">isFinite</span>(<span class="hljs-title class_">Infinity</span>);
<span class="hljs-comment">// false</span>
<span class="hljs-built_in">isFinite</span>(-<span class="hljs-title class_">Infinity</span>);
<span class="hljs-comment">//  alse</span>

<span class="hljs-comment">// true situaton</span>
<span class="hljs-built_in">isFinite</span>(<span class="hljs-number">0</span>);
<span class="hljs-comment">// true</span>
<span class="hljs-built_in">isFinite</span>(<span class="hljs-number">2e64</span>);
<span class="hljs-comment">// true</span>
<span class="hljs-built_in">isFinite</span>(<span class="hljs-string">&#x27;0&#x27;</span>);
<span class="hljs-comment">// true</span>

<span class="hljs-comment">// extraordinary</span>
<span class="hljs-title class_">Number</span>.<span class="hljs-built_in">isFinite</span>(<span class="hljs-literal">null</span>);
<span class="hljs-comment">// false</span>
<span class="hljs-title class_">Number</span>.<span class="hljs-built_in">isFinite</span>(<span class="hljs-string">&#x27;0&#x27;</span>);
<span class="hljs-comment">// false</span>
</code></pre>
<p>和全局的 <code>isFinite()</code> 函数相比，<code>Number.isFinite()</code> 不会强制将一个非数值的参数转换成数值，这就意味着，只有数值类型的值，且是有限值，才返回 <code>true</code>。</p>
