c:Te6b,<h1>isNaN</h1>
<p><code>isNaN()</code> 函数用于判断指定数字是否是非数字值 <code>NaN</code>。</p>
<p>该函数属于<code>Global</code>对象，所有主流浏览器均支持该函数。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-built_in">isNaN</span>(number);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>类型</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>number</code></td>
<td><code>Number</code> 类型</td>
<td>指定的数值</td>
</tr>
</tbody>
</table>
<p>⚠️ <strong>注意</strong>： 如果参数 <code>number</code> 不是 Number 类型，则 <code>isNaN()</code> 函数会将其强制转换为 Number 类型再进行判断。大多数其他类型的值无法强制转换为 Number 类型，则其转换结果为 <code>NaN</code>，即 <code>isNaN()</code> 函数返回 <code>true</code>。</p>
<ul>
<li><code>isNaN()</code> 函数的返回值是 Boolean 类型。
<ul>
<li>如果指定的数字为 <code>NaN</code>，则返回 <code>true</code></li>
<li>如果指定的数字为非 <code>NaN</code> 则返回 <code>false</code>。</li>
</ul>
</li>
</ul>
<h2>说明</h2>
<ul>
<li>通常使用此函数检测来自 <code>parseInt()</code> 和 <code>parseFloat()</code> 函数的返回值。</li>
<li>将某些不能强制转换为数字类型的值的非数字类型的值转换为数字类型的值时，也会得到 <code>NaN</code>。</li>
<li><code>NaN</code> 不能通过相等操作符来判断，因为 <code>NaN</code> 是唯一一个与其自身不等的值。</li>
</ul>
<h2>示例</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-built_in">isNaN</span>(<span class="hljs-title class_">NaN</span>);
<span class="hljs-comment">// true</span>
<span class="hljs-built_in">isNaN</span>(<span class="hljs-literal">undefined</span>);
<span class="hljs-comment">// true</span>

<span class="hljs-comment">// true</span>

<span class="hljs-built_in">isNaN</span>(<span class="hljs-literal">true</span>);
<span class="hljs-comment">// false</span>
<span class="hljs-built_in">isNaN</span>(<span class="hljs-literal">null</span>);
<span class="hljs-comment">// false</span>
<span class="hljs-built_in">isNaN</span>(<span class="hljs-number">37</span>);
<span class="hljs-comment">// false</span>

<span class="hljs-comment">// strings</span>
