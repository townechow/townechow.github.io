c:T6ce,<h1>decodeURIComponent</h1>
<p><strong><code>decodeURIComponent()</code></strong> 函数用于对统一资源标识符(URI)的一个已编码的组件进行解码，并返回其非编码形式。</p>
<p>该函数属于<code>Global</code>对象，所有主流浏览器均支持该函数。</p>
<blockquote>
<p>所谓的 URI 组件，就是 URI 的一部分，尤其是 URI 的参数部分。</p>
</blockquote>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-built_in">decodeURIComponent</span>(encodedURIString);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>类型</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>encodedURIString</code></td>
<td><code>String</code> 类型</td>
<td>已编码的 URI 字符串</td>
</tr>
</tbody>
</table>
<p>如果参数 <code>encodedURIString</code> 无效，将引发 URIError 错误。</p>
<p><code>decodeURIComponent()</code> 函数的返回值是 String 类型，返回一个已经解码的 URI 组件。</p>

<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> a = <span class="hljs-string">&#x27;Hello JavaScript&#x27;</span>;

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(c);
<span class="hljs-comment">// return &#x27;你好Javascript&#x27;</span>
</code></pre>
