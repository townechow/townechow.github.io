c:T4f7,<h1>String.fromCharCode</h1>
<p>静态 <code>String.fromCharCode()</code> 方法返回由指定的 UTF-16 代码单元序列创建的字符串。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-title class_">String</span>.<span class="hljs-title function_">fromCharCode</span>(num1 [, num2 [, num3 ...[, numN]]]);
</code></pre>
<p><strong>参数：</strong></p>
<ul>
<li><code>num</code>：一系列 UTF-16 代码单元的数字。范围介于 0 到 65535（<code>0xFFFF</code>）之间。大于 <code>0xFFFF</code> 的数字将被截断。不进行有效性检查。</li>
</ul>
<h2>使用示例</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-title class_">String</span>.<span class="hljs-title function_">fromCharCode</span>(<span class="hljs-number">65</span>, <span class="hljs-number">66</span>, <span class="hljs-number">67</span>);

<span class="hljs-comment">// -</span>
</code></pre>
