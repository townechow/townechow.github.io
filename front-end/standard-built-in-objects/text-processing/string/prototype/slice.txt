c:Tc22,<h1>String.prototype.slice()</h1>
<p><code>slice()</code> 方法提取字符串的某部分，并返回一个新的字符串。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js">str.<span class="hljs-title function_">slice</span>( startIndex [, endIndex] )
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
<th>类型</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>startIndex</code></td>
<td>指向字符串指定部分的开头的索引。</td>
<td>number</td>
</tr>
<tr>
<td><code>endIndex</code></td>
<td>可选，指向字符串指定部分的结尾的索引（不包括该索引），默认到字符串的结尾。</td>
<td>number</td>
</tr>
</tbody>
</table>
<p>返回一个从原字符串中提取出来的新字符串</p>
<h2>描述</h2>
<p><code>slice()</code> 函数一直从索引 <code>startIndex</code> 复制到 <code>endIndex</code> 所指示的字符，但是不包括 <code>endIndex</code> 索引上的字符。</p>
<ul>
<li>如果 <code>startIndex</code> 为负，则将其视为 <code>str.length + startIndex</code>。</li>
<li>如果省略 <code>endIndex</code>，则将一直提取到字符串的结尾。</li>
<li>如果 <code>endIndex</code> 为负，则将其视为 <code>str.length + endIndex</code>。</li>
<li>如果 <code>endIndex</code> 小于等于 <code>startIndex</code>，则不会复制任何字符，返回空字符串。</li>
</ul>
<h2>示例</h2>
<h3>代码示例</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> str = <span class="hljs-string">&#x27;abcdefghij&#x27;</span>;

<span class="hljs-comment">// 开始索引省略即从 0 开始提取，结束索引省略即提取到字符串末尾</span>
str.<span class="hljs-title function_">slice</span>(); <span class="hljs-comment">// &#x27;abcdefghij&#x27;</span>

<span class="hljs-comment">// 开始索引为 0，结束索引省略即提取到字符串末尾</span>
str.<span class="hljs-title function_">slice</span>(<span class="hljs-number">0</span>);
<span class="hljs-comment">// &#x27;abcdefghij&#x27;</span>

<span class="hljs-comment">// 开始索引为 0，结束索引为2</span>
str.<span class="hljs-title function_">slice</span>(<span class="hljs-number">0</span>, <span class="hljs-number">2</span>);
<span class="hljs-comment">// &#x27;ab&#x27;</span>

<span class="hljs-comment">// 开始索引为 -3 即负数，即为 -3+10=7，结束索引省略即提取到字符串末尾</span>
str.<span class="hljs-title function_">slice</span>(-<span class="hljs-number">3</span>);
<span class="hljs-comment">// &#x27;hij&#x27;</span>

<span class="hljs-comment">// 开始索引为 0，结束索引为 -3 即 -3+10=7</span>
str.<span class="hljs-title function_">slice</span>(<span class="hljs-number">0</span>, -<span class="hljs-number">3</span>);
<span class="hljs-comment">// &#x27;abcdef&#x27;</span>

<span class="hljs-comment">// 开始索引为 -3 即 -3+10=7，结束索引为 -1+10=9</span>
str.<span class="hljs-title function_">slice</span>(-<span class="hljs-number">3</span>, -<span class="hljs-number">1</span>);
</code></pre>
