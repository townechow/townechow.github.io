c:T16a5,<h1>String.prototype.substr()</h1>
<p><code>substr()</code> 函数用于返回当前字符串中一个连续的片段。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js">str.<span class="hljs-title function_">substr</span>( startIndex [, length] )
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
<th>类型</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>startIndex</code></td>
<td>指向字符串指定部分的开头的索引。</td>
<td>number</td>
</tr>
<tr>
<td><code>length</code></td>
<td>可选，返回的子字符串片段中包含的字符数。</td>
<td>number</td>
</tr>
</tbody>
</table>
<h2>描述</h2>
<p><code>substr()</code> 函数从 <code>str</code> 的索引 <code>startIndex</code> 处开始复制，直到复制 <code>length</code> 个字符或字符串的结尾为止。</p>
<ul>
<li>如果 <code>startIndex</code> 为正值，且大于或等于字符串的长度，则返回一个空字符串。</li>
<li>如果 <code>startIndex</code> 为负值，则将其视为 <code>str.length + startIndex</code> 开始的一个字符索引。若 <code>str.length + startIndex</code> 大于字符串的长度，则使用 0 作为开始提取的索引。</li>
<li>如果 <code>length</code> 为负数或 0，则不会复制任何字符，返回空字符串。</li>
<li>如果省略了 <code>length</code> 参数，则一直复制到字符串的结尾。</li>
</ul>
