c:T9ae,<h1>String.prototype.search()</h1>
<p><code>search()</code> 函数用于使用指定的正则表达式查找指定子字符串在当前字符串中第一次出现的位置。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js">str.<span class="hljs-title function_">search</span>(regexp);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
<th>类型</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>regexp</code></td>
<td>包含正则表达式模式的 <code>RegExp</code> 对象的实例。也可以是包含正则表达式模式的变量名或字符串。</td>
<td>RegExp</td>
</tr>
</tbody>
</table>
<p>如果参数 <code>regexp</code> 不是正则表达式对象（RegExp），而是字符串类型，则 <code>search()</code> 先将该字符串传递给 RegExp 的构造函数，将其转换为一个 RegExp 对象。</p>
<ul>
<li>如果匹配成功，则 <code>search()</code> 返回正则表达式在字符串中首次匹配项的<strong>索引</strong>。</li>
<li>如果没有查找到任何匹配，则返回 -1。</li>
</ul>
<h2>描述</h2>
<p>当你想要知道字符串中是否存在某个模式（pattern）时可使用 <code>search</code>，类似于正则表达式的 <code>test</code> 方法。当要了解更多匹配信息时，可使用 <code>match</code>（会更慢），该方法类似于正则表达式的 <code>exec</code> 方法。</p>
<h2>示例</h2>
<h3>代码示例</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> str = <span class="hljs-string">&#x27;Code123Player34Code456&#x27;</span>;

<span class="hljs-comment">// 查找2个连续数字的第一次出现</span>
str.<span class="hljs-title function_">search</span>(<span class="hljs-regexp">/\d{2}/</span>);
<span class="hljs-comment">// 4</span>

<span class="hljs-comment">// 该字符串等同于上一个正则表达式</span>
str.<span class="hljs-title function_">search</span>(<span class="hljs-string">&#x27;\\d{2}&#x27;</span>);
<span class="hljs-comment">// 4</span>

<span class="hljs-comment">// 查找不到匹配返回-1</span>
str.<span class="hljs-title function_">search</span>(<span class="hljs-regexp">/James/</span>);
<span class="hljs-comment">// -1</span>

<span class="hljs-comment">// 查找 &quot;player&quot;（带标志i，不区分大小写）</span>
str.<span class="hljs-title function_">search</span>(<span class="hljs-regexp">/player/i</span>);
<span class="hljs-comment">// 7</span>
</code></pre>
