c:Tc92,<h1>String.prototype.normalize()</h1>
<p><code>normalize()</code> 方法会按照指定的一种 Unicode 正规形式将当前字符串正规化。（如果该值不是字符串，则首先将其转换为一个字符串）。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js">str.<span class="hljs-title function_">normalize</span>([form]);
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
<th>类型</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>form</code></td>
<td>可选，四种 Unicode 正规形式（Unicode Normalization Form），默认值为 <code>NFC</code></td>
<td>string</td>
</tr>
</tbody>
</table>
<p>可选值：</p>
<ul>
<li><code>NFC</code>：Canonical Decomposition, followed by Canonical Composition</li>
<li><code>NFD</code>：Canonical Decomposition</li>
<li><code>NFKC</code>：Compatibility Decomposition, followed by Canonical Composition</li>
<li><code>NFKD</code>：Compatibility Decomposition</li>
</ul>
<p>返回给定字符串的 Unicode 规范化形式的字符串。</p>
<p>如果给 <code>form</code> 参数传入了上述四个字符串意外以外的参数，则会抛出 <code>RangeError</code> 异常。</p>
<h2>示例</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-comment">// U+1E9B: LATIN SMALL LETTER LONG S WITH DOT ABOVE</span>
<span class="hljs-comment">// U+0323: COMBINING DOT BELOW</span>
<span class="hljs-keyword">var</span> str = <span class="hljs-string">&#x27;\u1E9B\u0323&#x27;</span>;

<span class="hljs-comment">// Canonically-composed form (NFC)</span>

<span class="hljs-comment">// U+1E9B: LATIN SMALL LETTER LONG S WITH DOT ABOVE</span>
<span class="hljs-comment">// U+0323: COMBINING DOT BELOW</span>
str.<span class="hljs-title function_">normalize</span>(<span class="hljs-string">&#x27;NFC&#x27;</span>); <span class="hljs-comment">// &quot;\u1E9B\u0323&quot;</span>
str.<span class="hljs-title function_">normalize</span>(); <span class="hljs-comment">// same as above</span>

<span class="hljs-comment">// Canonically-decomposed form (NFD)</span>

<span class="hljs-comment">// U+017F: LATIN SMALL LETTER LONG S</span>
<span class="hljs-comment">// U+0323: COMBINING DOT BELOW</span>
<span class="hljs-comment">// U+0307: COMBINING DOT ABOVE</span>
str.<span class="hljs-title function_">normalize</span>(<span class="hljs-string">&#x27;NFD&#x27;</span>); <span class="hljs-comment">// &quot;\u017F\u0323\u0307&quot;</span>

<span class="hljs-comment">// Compatibly-composed (NFKC)</span>

<span class="hljs-comment">// U+1E69: LATIN SMALL LETTER S WITH DOT BELOW AND DOT ABOVE</span>
str.<span class="hljs-title function_">normalize</span>(<span class="hljs-string">&#x27;NFKC&#x27;</span>); <span class="hljs-comment">// &quot;\u1E69&quot;</span>

<span class="hljs-comment">// Compatibly-decomposed (NFKD)</span>

<span class="hljs-comment">// U+0073: LATIN SMALL LETTER S</span>
<span class="hljs-comment">// U+0323: COMBINING DOT BELOW</span>
<span class="hljs-comment">// U+0307: COMBINING DOT ABOVE</span>
str.<span class="hljs-title function_">normalize</span>(<span class="hljs-string">&#x27;NFKD&#x27;</span>); <span class="hljs-comment">// &quot;\u0073\u0323\u0307&quot;</span>
</code></pre>
