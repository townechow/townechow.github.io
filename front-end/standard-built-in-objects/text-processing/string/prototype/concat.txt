c:T823,<h1>String.prototype.concat()</h1>
<p><code>concat()</code> 函数用于将当前字符串与指定字符串进行拼接，并返回拼接后的字符串。</p>
<h2>语法</h2>
<pre class="hljs language-js"  style=--lang:"js" ><code>str.<span class="hljs-title function_">concat</span>( string2, string3 [, ..., stringN] )
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
<th>类型</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>str2...strN</code></td>
<td>和原字符串连接的多个字符串。</td>
<td>string</td>
</tr>
</tbody>
</table>
<p><code>concat()</code> 函数的返回值为 <code>String</code> 类型，其返回值为拼接后的字符串。</p>
<p><code>concat()</code> 函数的作用等同于运算符 <code>+</code>，例如：<code>str.concat(str1, str2)</code> 等同于 <code>str + str1 + str2</code>。</p>
<h2>说明</h2>
<p><code>concat</code> 方法将一个或多个字符串与原字符串连接合并，形成一个新的字符串并返回。 <code>concat</code> 方法并不影响原字符串。</p>
<h2>示例</h2>
<p>下面的例子演示如何将多个字符串与原字符串合并为一个新字符串</p>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-keyword">var</span> hello = <span class="hljs-string">&#x27;Hello, &#x27;</span>;

hello.<span class="hljs-title function_">concat</span>(<span class="hljs-string">&#x27;Kevin&#x27;</span>, <span class="hljs-string">&#x27; have a nice day.&#x27;</span>);
<span class="hljs-comment">// Hello, Kevin have a nice day.</span>
</code></pre>
<p>建议使用赋值操作符（<code>+</code> 或 <code>+=</code>）代替字符串的 <code>concat</code> 方法。</p>
<pre class="hljs language-js"  style=--lang:"js" ><code><span class="hljs-keyword">var</span> hello = <span class="hljs-string">&#x27;Hello, &#x27;</span>;

hello += <span class="hljs-string">&#x27;world&#x27;</span>;

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(hello);
<span class="hljs-comment">// Hello, world</span>
</code></pre>
