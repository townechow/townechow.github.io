c:T7a8,<h1>String.prototype.trim()</h1>
<p><code>trim()</code> 方法会从一个字符串的两端删除空白字符。在这个上下文中的空白字符是所有的空白字符（space, tab, no-break space 等）以及所有行终止符字符（如 <code>LF</code>，<code>CR</code>）。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js">str.<span class="hljs-title function_">trim</span>();
</code></pre>
<p><code>trim()</code> 方法并不影响原字符串本身，它返回的是一个新的字符串。</p>
<h2>示例</h2>
<h3>代码示例</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> foo = <span class="hljs-string">&#x27;   foo  &#x27;</span>;

foo.<span class="hljs-title function_">trim</span>();
<span class="hljs-comment">// &#x27;foo&#x27;</span>
</code></pre>
<p>另一个 <code>trim()</code> 例子，只从一边删除</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> bar = <span class="hljs-string">&#x27;bar    &#x27;</span>;

bar.<span class="hljs-title function_">trim</span>();
<span class="hljs-comment">// &#x27;bar&#x27;</span>
</code></pre>
