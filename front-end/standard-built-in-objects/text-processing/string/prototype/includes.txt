c:Teef,<h1>String.prototype.includes()</h1>
<p><code>includes()</code> 方法用于判断一个字符串是否包含在另一个字符串中，根据情况返回 <code>true</code> 或 <code>false</code>。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js">str.<span class="hljs-title function_">includes</span>( searchString [, startIndex] )
</code></pre>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
<th>类型</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>searchString</code></td>
<td>要在字符串中搜索的字符串</td>
<td>string</td>
</tr>
<tr>
<td><code>startIndex</code></td>
<td>从当前字符串指定索引位置开发搜索子字符串，默认为 0，包含该索引</td>
<td>number</td>
</tr>
</tbody>
</table>
<p>这个方法可以帮你判断一个字符串是否包含另外一个字符串。</p>
<p>这个方法搜索匹配的字符串是区分大小写的。</p>
