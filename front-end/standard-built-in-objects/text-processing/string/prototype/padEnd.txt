c:T12b6,<h1>String.prototype.padEnd()</h1>
<p>⭐️ <code>ES2017(ES8)新特性</code></p>
<p><code>String.prototype.padEnd()</code> 方法会用一个字符串填充当前字符串（如果需要的话则重复填充），返回填充后达到指定长度的字符串。从当前字符串的末尾（右侧）开始填充。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">str.<span class="hljs-title function_">padEnd</span>(maxLength [, fillString]);
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">String</span> {
  <span class="hljs-title function_">padEnd</span>(<span class="hljs-attr">maxLength</span>: <span class="hljs-built_in">number</span>, <span class="hljs-attr">fillString</span>?: <span class="hljs-built_in">string</span>): <span class="hljs-built_in">string</span>;
}
</code></pre>
<p>参数说明：</p>
<table>
<thead>
<tr>
<th style="text-align:left">参数</th>
<th style="text-align:left">说明</th>
<th style="text-align:left">类型</th>
</tr>
</thead>
<tbody>
<tr>
<td style="text-align:left"><code>maxLength</code></td>
<td style="text-align:left">当前字符串需要填充到的目标长度。如果这个数值小于当前字符串的长度，则返回当前字符串本身。</td>
<td style="text-align:left">number</td>
</tr>
<tr>
<td style="text-align:left"><code>fillString</code></td>
<td style="text-align:left">（可选）填充字符串。如果字符串太长，使填充后的字符串长度超过了目标长度，则只保留最左侧的部分，其他部分会被截断。</td>
<td style="text-align:left">string</td>
</tr>
</tbody>
</table>
