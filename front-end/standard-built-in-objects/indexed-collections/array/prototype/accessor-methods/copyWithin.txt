c:Tad8,<h1>Array.prototype.copyWithin()</h1>
<p><code>Array.prototype.copyWithin()</code> 方法浅复制数组的一部分到同一数组中的另一个位置，并返回它，不会改变原数组的长度。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">arr.<span class="hljs-title function_">copyWithin</span>( target [, start [, end]]);
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  <span class="hljs-title function_">copyWithin</span>(<span class="hljs-attr">target</span>: <span class="hljs-built_in">number</span>, <span class="hljs-attr">start</span>: <span class="hljs-built_in">number</span>, <span class="hljs-attr">end</span>?: <span class="hljs-built_in">number</span>): <span class="hljs-variable language_">this</span>;
}
</code></pre>
<p>参数说明：</p>

<td style="text-align:left">target</td>
<td style="text-align:left">0 为基底的索引，复制序列到该位置。如果是负数，<code>target</code> 将从末尾开始计算。如果 <code>target</code> 大于等于数组长度，将不会发生拷贝。如果 <code>target</code> 在 <code>start</code> 之后，复制的序列将被修改以复合数组长度</td>
<td style="text-align:left">number</td>

<td style="text-align:left">number</td>

<td style="text-align:left">number</td>
