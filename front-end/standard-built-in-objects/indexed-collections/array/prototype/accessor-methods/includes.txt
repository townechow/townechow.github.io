c:T1b6c,<h1>Array.prototype.includes()</h1>
<p>⭐️ <code>ES2016(ES7)新特性</code></p>
<p><code>Array.prototype.includes()</code> 方法用于判断数组是否包含指定的值。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">arr.<span class="hljs-title function_">includes</span>(searchElement [, fromIndex])
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  <span class="hljs-title function_">includes</span>(<span class="hljs-attr">searchElement</span>: T, <span class="hljs-attr">fromIndex</span>?: <span class="hljs-built_in">number</span>): <span class="hljs-built_in">boolean</span>;
}
</code></pre>
<p>参数说明：</p>
<table>
<thead>
<tr>
<th style="text-align:left">参数</th>
<th style="text-align:left">说明</th>
<th style="text-align:left">类型</th>
</tr>
</thead>
<tbody>
<tr>
<td style="text-align:left">searchElement</td>
<td style="text-align:left">需要查找的元素值</td>
<td style="text-align:left">any</td>
</tr>
<tr>
<td style="text-align:left">fromIndex</td>
<td style="text-align:left">查找数组开始的索引</td>
<td style="text-align:left">number</td>
</tr>
</tbody>
</table>
<p>返回值：</p>
<p>如果存在指定值则返回 <code>true</code>，否则返回 <code>false</code>。</p>
<h2>代码示例</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">const</span> arr = [<span class="hljs-number">1</span>, <span class="hljs-number">2</span>, <span class="hljs-number">3</span>];

arr.<span class="hljs-title function_">includes</span>(<span class="hljs-number">2</span>);
<span class="hljs-comment">// true</span>

arr.<span class="hljs-title function_">includes</span>(<span class="hljs-number">4</span>);
<span class="hljs-comment">// false</span>

arr.<span class="hljs-title function_">includes</span>(<span class="hljs-number">3</span>, <span class="hljs-number">3</span>);
<span class="hljs-comment">// false</span>

arr.<span class="hljs-title function_">includes</span>(<span class="hljs-number">3</span>, -<span class="hljs-number">1</span>);
<span class="hljs-comment">// true</span>

<span class="hljs-keyword">const</span> arr2 = [<span class="hljs-number">1</span>, <span class="hljs-number">2</span>, <span class="hljs-title class_">NaN</span>];

arr2.<span class="hljs-title function_">includes</span>(<span class="hljs-title class_">NaN</span>);
<span class="hljs-comment">// true</span>
</code></pre>
<h3>参数值为 NaN</h3>
<p><code>Array.prototype.indexOf()</code> 无法查找出数组中的 NaN，而该方法可以判断出数组中是否包含 NaN。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">const</span> arr = [<span class="hljs-number">1</span>, <span class="hljs-number">2</span>, <span class="hljs-number">3</span>, <span class="hljs-number">4</span>, <span class="hljs-title class_">NaN</span>, <span class="hljs-number">6</span>, <span class="hljs-title class_">NaN</span>];

arr.<span class="hljs-title function_">includes</span>(<span class="hljs-title class_">NaN</span>);
<span class="hljs-comment">// true</span>
</code></pre>
<h3>开始索引超限</h3>
<p>如果 <code>fromIndex</code> 大于等于 <strong>数组长度</strong> ，则返回 <code>false</code> 。该数组不会被搜索。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> arr = [<span class="hljs-string">&#x27;a&#x27;</span>, <span class="hljs-string">&#x27;b&#x27;</span>, <span class="hljs-string">&#x27;c&#x27;</span>];

arr.<span class="hljs-title function_">includes</span>(<span class="hljs-string">&#x27;c&#x27;</span>, <span class="hljs-number">3</span>);
<span class="hljs-comment">// false</span>
arr.<span class="hljs-title function_">includes</span>(<span class="hljs-string">&#x27;c&#x27;</span>, <span class="hljs-number">100</span>);
<span class="hljs-comment">// false</span>
</code></pre>
<h3>开始索引为负值</h3>
<p>如果 <code>fromIndex</code> 为负值，计算出的索引将作为开始搜索 <code>searchElement</code> 的位置。如果计算出的索引小于 0，则整个数组都会被搜索。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-comment">// 数组长度是 3</span>
<span class="hljs-comment">// fromIndex 是 -100</span>
<span class="hljs-comment">// computed index 是 3 + (-100) = -97</span>

<span class="hljs-keyword">var</span> arr = [<span class="hljs-string">&#x27;a&#x27;</span>, <span class="hljs-string">&#x27;b&#x27;</span>, <span class="hljs-string">&#x27;c&#x27;</span>];

arr.<span class="hljs-title function_">includes</span>(<span class="hljs-string">&#x27;a&#x27;</span>, -<span class="hljs-number">100</span>);
<span class="hljs-comment">// true</span>
arr.<span class="hljs-title function_">includes</span>(<span class="hljs-string">&#x27;b&#x27;</span>, -<span class="hljs-number">100</span>);
<span class="hljs-comment">// true</span>
arr.<span class="hljs-title function_">includes</span>(<span class="hljs-string">&#x27;c&#x27;</span>, -<span class="hljs-number">100</span>);
<span class="hljs-comment">// true</span>
</code></pre>
<h3>类数组通用方法</h3>
<p><code>Array.prototype.includes()</code> 方法有意设计为通用方法。它不要求 <code>this</code> 值是数组对象，所以它可以被用于其他类型的对象（比如类数组对象）。</p>
<p>下面的例子展示了 在函数的 <code>arguments</code> 对象上调用的 <code>includes()</code> 方法。</p>

  <span class="hljs-comment">// false</span>
})(<span class="hljs-string">&#x27;a&#x27;</span>, <span class="hljs-string">&#x27;b&#x27;</span>, <span class="hljs-string">&#x27;c&#x27;</span>);
