c:T1175,<h1>Array.prototype.indexOf()</h1>
<p><code>Array.prototype.indexOf()</code> 方法用于查找数组成员第一次出现指定字符的位置。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">arr.<span class="hljs-title function_">indexOf</span>( searchElement [, fromIndex] )
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  <span class="hljs-title function_">indexOf</span>(<span class="hljs-attr">searchElement</span>: T, <span class="hljs-attr">fromIndex</span>?: <span class="hljs-built_in">number</span>): <span class="hljs-built_in">number</span>;
}
</code></pre>
<p>参数说明：</p>
<table>
<thead>
<tr>
<th style="text-align:left">参数</th>
<th style="text-align:left">说明</th>
<th style="text-align:left">类型</th>
</tr>
</thead>
<tbody>
<tr>
<td style="text-align:left"><code>searchElement</code></td>
<td style="text-align:left">要查找的数组元素</td>
<td style="text-align:left">any</td>
</tr>
<tr>
<td style="text-align:left"><code>fromIndex</code></td>
<td style="text-align:left">可选，在当前字符串中查找的起始索引，默认为 0</td>
<td style="text-align:left">number</td>
</tr>
</tbody>
</table>
<p>返回值：</p>
<p>返回数组元素在当前数组中第一次查找到的起始位置（索引）</p>
<h2>方法说明</h2>
<p>该方法使用 Strict Equality（无论是绝对相等 <code>===</code>，还是 Triple-equals 操作符都基于同样的方法）进行判断查找的元素与数组中包含的元素之间的关系。</p>
<h2>代码示例</h2>
<h3>基本用法</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> arr = [<span class="hljs-number">1</span>, <span class="hljs-number">2</span>, <span class="hljs-number">3</span>, <span class="hljs-number">4</span>, <span class="hljs-number">5</span>];

arr.<span class="hljs-title function_">indexOf</span>(<span class="hljs-number">1</span>);

<span class="hljs-comment">// 3</span>
arr.<span class="hljs-title function_">indexOf</span>(<span class="hljs-number">3</span>, -<span class="hljs-number">1</span>);
<span class="hljs-comment">// -1</span>
arr.<span class="hljs-title function_">indexOf</span>(<span class="hljs-number">3</span>, -<span class="hljs-number">3</span>);
<span class="hljs-comment">// 2</span>
</code></pre>

<pre class="hljs"><code class="language-js"><span class="hljs-comment">// 存放指定元素出现的位置的数组</span>
