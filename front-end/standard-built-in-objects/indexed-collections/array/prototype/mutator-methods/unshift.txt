c:Tc2c,<h1>Array.prototype.unshift()</h1>
<p><code>Array.prototype.unshift()</code> 方法用于向当前数组的开头位置插入一个或多个指定的元素，并返回插入后的数组长度。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">arr.<span class="hljs-title function_">unshift</span>( item1 [, items... ] )
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  <span class="hljs-title function_">unshif</span>(...<span class="hljs-attr">items</span>: T[]): <span class="hljs-built_in">number</span>;
}
</code></pre>
<p>参数说明：</p>
<table>
<thead>
<tr>
<th style="text-align:left">参数</th>
<th style="text-align:left">说明</th>
<th style="text-align:left">类型</th>
</tr>
</thead>
<tbody>
<tr>
<td style="text-align:left">item1</td>
<td style="text-align:left">添加到当前数组开头处的元素</td>
<td style="text-align:left">any</td>
</tr>
<tr>
<td style="text-align:left">itemN</td>
<td style="text-align:left">要添加到当前数组开头处的其他项，可以有多个。</td>
<td style="text-align:left">any</td>
</tr>
</tbody>
</table>
<p>返回值：</p>
<p>返回插入元素后的当前数组的长度。</p>
<h2>方法说明</h2>
<p>当向数组中添加新的元素时，数组的 <code>length</code> 属性也会随之改变（如果数组中有元素的话），一般而言，数组的 <code>length</code> 属性将会加 <code>N</code> （ <code>N</code> 为添加的元素个数）。</p>
<h2>代码示例</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">let</span> arr = [<span class="hljs-number">1</span>, <span class="hljs-number">2</span>];

arr.<span class="hljs-title function_">unshift</span>(<span class="hljs-number">0</span>);
<span class="hljs-comment">// 3</span>

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(arr);
<span class="hljs-comment">// [0, 1, 2]</span>

arr.<span class="hljs-title function_">unshift</span>(-<span class="hljs-number">2</span>, -<span class="hljs-number">1</span>);
<span class="hljs-comment">// 5</span>

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(arr);
<span class="hljs-comment">// [-2, -1, 0, 1, 2]</span>

arr.<span class="hljs-title function_">unshift</span>([-<span class="hljs-number">3</span>]);
<span class="hljs-comment">// 6</span>

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(arr);
<span class="hljs-comment">// [[-3], -2, -1, 0, 1, 2]</span>
</code></pre>
