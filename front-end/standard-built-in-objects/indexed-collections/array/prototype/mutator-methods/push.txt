c:T1184,<h1>Array.prototype.push()</h1>
<p><code>Array.prototype.push()</code> 方法用于向当前数组的末尾添加一个或多个元素，并返回新的数组长度。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">arr.<span class="hljs-title function_">push</span>( item1 [, items...] )
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  <span class="hljs-title function_">push</span>(...<span class="hljs-attr">items</span>: T[]): <span class="hljs-built_in">number</span>;
}
</code></pre>
<p>参数说明：</p>
<table>
<thead>
<tr>
<th style="text-align:left">参数</th>
<th style="text-align:left">说明</th>
<th style="text-align:left">类型</th>
</tr>
</thead>
<tbody>
<tr>
<td style="text-align:left">item1</td>
<td style="text-align:left">添加元素</td>
<td style="text-align:left">any</td>
</tr>
<tr>
<td style="text-align:left">itemN</td>
<td style="text-align:left">添加的其他元素</td>
<td style="text-align:left">any</td>
</tr>
</tbody>
</table>
<p>如果添加的元素类型为数组类型，仍然会被当作一个元素看待，只是这个元素是数组类型而已。如果要合并两个数组，请使用 <code>concat()</code> 函数。</p>
<p>返回值：</p>
<p>返回添加元素后的数组长度。</p>
<h2>方法说明</h2>
<p>当向数组中添加新的元素时，数组的 <code>length</code> 属性也会随之改变。一般而言，数组的 <code>length</code> 属性将会加 <code>N</code> （ <code>N</code> 为添加的元素个数）。</p>
<p><code>push()</code> 方法有意具有通用性。该方法和 <code>call()</code> 或 <code>apply()</code> 一起使用时，可应用在类似数组的对象上。<code>push()</code> 方法根据 length 属性来决定从哪里开始插入给定的值。如果 <code>length</code> 不能被转成一个数值，则插入的元素索引为 0，包括 <code>length</code> 不存在时。当 <code>length</code> 不存在时，将会创建它。</p>

<h3>基本用法</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">const</span> foo = [<span class="hljs-string">&#x27;a&#x27;</span>, <span class="hljs-string">&#x27;b&#x27;</span>];

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(foo);
<span class="hljs-comment">// [&#x27;a&#x27;, &#x27;b&#x27;, &#x27;c&#x27;, &#x27;d&#x27;]</span>

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(bar);
<span class="hljs-comment">// 4</span>
</code></pre>
<h3>合并数组</h3>
<p>该示例使用 <code>apply()</code> 添加第二个数组的所有元素。注意当第二个数组太大时不要使用这个方法来合并数组，因为事实上一个函数能够接受的参数个数是有限制的。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">const</span> foo = [<span class="hljs-string">&#x27;a&#x27;</span>, <span class="hljs-string">&#x27;b&#x27;</span>];

<span class="hljs-title class_">Array</span>.<span class="hljs-property"><span class="hljs-keyword">prototype</span></span>.<span class="hljs-property">push</span>.<span class="hljs-title function_">apply</span>(foo, bar);

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(foo);
