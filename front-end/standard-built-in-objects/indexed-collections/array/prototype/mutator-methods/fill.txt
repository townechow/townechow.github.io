c:T160a,<h1>Array.prototype.fill()</h1>
<p><code>Array.prototype.fill()</code> 方法用于将一个固定值填充到数组中从起始索引到终止索引内的全部元素。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">arr.<span class="hljs-title function_">fill</span>( value [, start [, end] ] )
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  <span class="hljs-title function_">fill</span>(<span class="hljs-attr">value</span>: T, <span class="hljs-attr">start</span>?: <span class="hljs-built_in">number</span>, <span class="hljs-attr">end</span>?: <span class="hljs-built_in">number</span>): <span class="hljs-variable language_">this</span>;
}
</code></pre>
<p>参数说明：</p>
<table>
<thead>
<tr>
<th style="text-align:left">参数</th>
<th style="text-align:left">说明</th>
<th style="text-align:left">类型</th>
</tr>
</thead>
<tbody>
<tr>
<td style="text-align:left">value</td>
<td style="text-align:left">填充数组元素的值</td>
<td style="text-align:left">any</td>
</tr>
<tr>
<td style="text-align:left">start</td>
<td style="text-align:left">起始索引，默认为 0。</td>
<td style="text-align:left">number</td>
</tr>
<tr>
<td style="text-align:left">end</td>
<td style="text-align:left">结束索引，默认为 <code>arr.length</code> 。</td>
<td style="text-align:left">number</td>
</tr>
</tbody>
</table>
<h2>方法说明</h2>
<p>具体填充区间始于 <code>start</code>，结束但不包括于 <code>emdIndex</code>（半开半闭区间）</p>
<ul>
<li>当 <code>start</code> 为负数，则开始索引为 <code>arr.length + start</code></li>
<li>当 <code>end</code> 为负数，则结束索引为 <code>arr.length + end</code></li>
</ul>
