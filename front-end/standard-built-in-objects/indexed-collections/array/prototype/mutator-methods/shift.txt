c:T9df,<h1>Array.prototype.shift()</h1>
<p><code>Array.prototype.shift()</code> 方法用于移除数组第一个成员，并返回移除的元素。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">arr.<span class="hljs-title function_">shift</span>();
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  <span class="hljs-title function_">shift</span>(): T | <span class="hljs-literal">undefined</span>;
}
</code></pre>
<p>返回值：</p>
<p>返回被移除的数组成员。如果该数组为空（没有任何元素），则返回 <code>undefined</code>。</p>
<h2>方法说明</h2>
<p>由于本函数会移除数组中的第一个元素，数组的 <code>length</code> 属性也会随之改变（如果数组中有元素的话），一般而言，数组的 <code>length</code> 属性将会减 1。</p>

<h3>基本用法</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">let</span> foo = [<span class="hljs-string">&#x27;a&#x27;</span>, <span class="hljs-string">&#x27;b&#x27;</span>, <span class="hljs-string">&#x27;c&#x27;</span>, <span class="hljs-string">&#x27;d&#x27;</span>];

<span class="hljs-keyword">let</span> bar = foo.<span class="hljs-title function_">shift</span>();

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(foo);

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(bar);
<span class="hljs-comment">// &#x27;a&#x27;</span>
</code></pre>
