c:T1e85,<h1>Array.prototype.find()</h1>
<p><code>Array.prototype.find()</code> 方法返回数组中满足提供的判定函数的第一个成员。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs"><code class="language-js">arr.<span class="hljs-title function_">find</span>( callback [, thisArg ] )
</code></pre>
<p>类型声明：</p>
<pre class="hljs"><code class="language-ts"><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  find&lt;S <span class="hljs-keyword">extends</span> T&gt;(
    <span class="hljs-attr">predicate</span>: <span class="hljs-function">(<span class="hljs-params"><span class="hljs-attr">this</span>: <span class="hljs-built_in">void</span>, <span class="hljs-attr">value</span>: T, <span class="hljs-attr">index</span>: <span class="hljs-built_in">number</span>, <span class="hljs-attr">obj</span>: T[]</span>) =&gt;</span> value is S,
    <span class="hljs-attr">thisArg</span>?: <span class="hljs-built_in">any</span>
  ): S | <span class="hljs-literal">undefined</span>;
  <span class="hljs-title function_">find</span>(<span class="hljs-attr">predicate</span>: <span class="hljs-function">(<span class="hljs-params"><span class="hljs-attr">value</span>: T, <span class="hljs-attr">index</span>: <span class="hljs-built_in">number</span>, <span class="hljs-attr">obj</span>: T[]</span>) =&gt;</span> <span class="hljs-built_in">unknown</span>, <span class="hljs-attr">thisArg</span>?: <span class="hljs-built_in">any</span>): T | <span class="hljs-literal">undefined</span>;
}
</code></pre>
<p>参数说明：</p>

<p><code>callback</code> 函数的参数：</p>
<ul>
<li><code>currentValue</code>：当前数组中处理的元素</li>
<li><code>index</code>：数组中正处理的当前元素的索引</li>
<li><code>array</code>：被调用的数组</li>
</ul>
<p>返回值：</p>
<p>当遍历到的数组成员通过回调函数的判定时，返回数组中该成员，否则返回 <code>undefined</code>。</p>
