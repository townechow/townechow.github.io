c:T12b5,<h1>Array.prototype.findIndex()</h1>
<p><code>findIndex()</code>方法返回数组中满足提供的测试函数的<strong>第一个元素</strong>的<strong>索引</strong>。否则返回-1。</p>
<h2>语法</h2>
<p>语法：</p>
<pre class="hljs language-js"  style=--lang:"js" ><code>arr.<span class="hljs-title function_">findIndex</span>( callback [, thisArg ])
</code></pre>
<p>类型声明：</p>
<pre class="hljs language-ts"  style=--lang:"ts" ><code><span class="hljs-keyword">interface</span> <span class="hljs-title class_">Array</span>&lt;T&gt; {
  <span class="hljs-title function_">findIndex</span>(<span class="hljs-attr">predicate</span>: <span class="hljs-function">(<span class="hljs-params"><span class="hljs-attr">value</span>: T, <span class="hljs-attr">index</span>: <span class="hljs-built_in">number</span>, <span class="hljs-attr">obj</span>: T[]</span>) =&gt;</span> <span class="hljs-built_in">unknown</span>, <span class="hljs-attr">thisArg</span>?: <span class="hljs-built_in">any</span>): <span class="hljs-built_in">number</span>;
}
</code></pre>
