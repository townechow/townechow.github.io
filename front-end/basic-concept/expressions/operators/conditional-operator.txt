c:Tcd6,<h1>条件运算符</h1>
<p><strong>条件运算符（Conditional Operator）<strong>是 JavaScript 中唯一的一个</strong>三元运算符</strong>（三个操作数），有时直接称做三元运算符。</p>
<pre class="hljs"><code class="language-js">variable = boolean_expression ? true_value : false_value;
</code></pre>
<p>本质上，这就是基于对 <code>boolean_expression</code> 求值的结果，决定给变量 <code>variable</code> 赋什么值。如果求值结果是 <code>true</code> ，则给变量 <code>variable</code> 赋值<code>true_value</code>；如果求值结果是 <code>false</code>，则给变量 <code>variable</code> 赋值<code>false_value</code>。</p>
<p>条件运算符的操作数可以是任意类型，第一个操作数当成布尔值，如果它是真值 <code>true</code>，那么将计算第二个操作数，并返回其计算结果。否则，如果第一个操作数是假值 <code>false</code>，那么将计算第三个操作数，并返回其计算结果。第二个和第三个操作数总是会计算其中之一，不可能两者同时执行。</p>
<p>其实使用 <code>if</code> 语句也会带来同样的效果，<code>?:</code>运算符只是提供了一种简写形式。下面是一个 <code>?:</code> 的典型应用场景，判断一个变量是否有定义(并拥有一个有意义的真值)，如果有定义则使用它，如果无定义则使用一个默认值:</p>
<pre class="hljs"><code class="language-js">greeting = <span class="hljs-string">&#x27;hello &#x27;</span> + (username ? username : <span class="hljs-string">&#x27;there&#x27;</span>);
</code></pre>
<p>这和下面使用 <code>if</code> 语句的代码是等价的，但显然上面的代码更加简洁:</p>
<pre class="hljs"><code class="language-js">greeting = <span class="hljs-string">&#x27;hello &#x27;</span>;
<span class="hljs-keyword">if</span> (username) greeting += username;
<span class="hljs-keyword">else</span> greeting += <span class="hljs-string">&#x27;there&#x27;</span>;
</code></pre>
<p>条件运算符（三元条件表达式）与 <code>if...else</code> 语句具有同样表达效果，但是两者有一个重大差别。</p>
<table>
<thead>
<tr>
<th style="text-align:left">项</th>
<th style="text-align:left">类型</th>
<th style="text-align:left">返回值</th>
</tr>

<td style="text-align:left">语句</td>
<td style="text-align:left">无</td>
</tr>

</tr>

</table>
<p>因此，在需要返回值的场合，只能使用条件运算符（三元条件表达式），而不能使用 <code>if...else</code>。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(<span class="hljs-literal">true</span> ? <span class="hljs-string">&#x27;T&#x27;</span> : <span class="hljs-string">&#x27;F&#x27;</span>);
<span class="hljs-comment">// &#x27;T&#x27;</span>
</code></pre>
<p>上面代码中，<code>console.log()</code>方法的参数必须是一个表达式，这时就只能使用三元条件表达式。</p>
