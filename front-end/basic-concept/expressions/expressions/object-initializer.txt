c:Tac0,<h1>对象初始化表达式</h1>
<p>对象和数组初始化表达式实际上是一个新创建的对象和数组。这些初始化表达式有时称作 <strong>对象直接量</strong> 和 <strong>数组直接量</strong>。然而和布尔值直接量不同，它们实际上不是原始表达式，因为它们所包含的成员或者元素都是子表达式。</p>
<p>对象初始化表达式和数组初始化表达式非常相似，只是方括号被花括号代替，并且每个子表达式都包含一个属性名和一个冒号作为前缀。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-comment">// 一个拥有两个属性成员的对象</span>
<span class="hljs-keyword">var</span> p = {
  <span class="hljs-attr">x</span>: <span class="hljs-number">2.3</span>,
  <span class="hljs-attr">y</span>: -<span class="hljs-number">1.2</span>,
};

<span class="hljs-comment">// 一个空对象</span>
<span class="hljs-keyword">var</span> q = {};

<span class="hljs-comment">// q 的属性成员和 p 的一样</span>
q.<span class="hljs-property">x</span> = <span class="hljs-number">2.3</span>;
q.<span class="hljs-property">y</span> = -<span class="hljs-number">1.2</span>;
</code></pre>
<p>对象直接量也可以嵌套。</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> rectangle = {
  <span class="hljs-attr">upperLeft</span>: { <span class="hljs-attr">x</span>: <span class="hljs-number">2</span>, <span class="hljs-attr">y</span>: <span class="hljs-number">2</span> },
  <span class="hljs-attr">lowRight</span>: { <span class="hljs-attr">x</span>: <span class="hljs-number">4</span>, <span class="hljs-attr">y</span>: <span class="hljs-number">5</span> },
};
</code></pre>
<p>JavaScript 求对象初始化表达式的值的时候，对象表达式也都会各自计算一次，并且它们不必包含常数值：它们可以是任意 JavaScript 表达式。</p>
<p>同样，对喜爱那个直接量中的属性名称可以是字符串而不是标识符（这在那些只能使用保留字或一些非法标识符作为属性名的地方非常有用）</p>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> side = <span class="hljs-number">1</span>;
<span class="hljs-keyword">var</span> square = {
  <span class="hljs-attr">upperLeft</span>: {
    <span class="hljs-attr">x</span>: p.<span class="hljs-property">x</span>,
    <span class="hljs-attr">y</span>: p.<span class="hljs-property">y</span>,
  },
  <span class="hljs-attr">lowerRight</span>: {
    <span class="hljs-attr">x</span>: p.<span class="hljs-property">x</span> + side,
    <span class="hljs-attr">y</span>: p.<span class="hljs-property">y</span> + side,
  },
};
</code></pre>
