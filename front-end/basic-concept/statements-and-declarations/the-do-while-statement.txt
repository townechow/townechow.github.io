c:T912,<h1>do...while 语句</h1>
<p><strong>do...while 语句</strong>创建一个执行指定语句的循环，直到 <code>condition</code> 值为 false。在执行 <code>statement</code> 后检测 <code>condition</code>，所以指定的 <code>statement</code> 至少执行一次。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">do</span> {
  statement;
} <span class="hljs-keyword">while</span> (expression);
</code></pre>
<h3>参数</h3>
<table>
<thead>
<tr>
<th>参数</th>
<th>描述</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>statement</code></td>
<td>执行至少一次的语句，并在每次 <code>condition</code> 值为真时重新执行。想执行多行语句，可使用 <code>block</code>语句（<code>{ ... }</code>）包裹这些语句。</td>
</tr>
<tr>
<td><code>expression</code></td>
<td>循环中每次都会计算的表达式。如果 <code>condition</code> 值为 <code>true</code>，<code>statement</code> 会再次执行。当 <code>condition</code> 值为 <code>false</code>，则跳到 <code>do...while</code> 之后的语句。</td>
</tr>
</tbody>
</table>
<h3>描述</h3>
<ul>
<li>像 <code>do-while</code> 这种后测试循环语句最常用于循环体中的代码至少要被执行一次的情形。</li>
<li><code>do/while</code> 循环和普通 <code>while</code> 循环之间有两点语法方面的不同之处。
<ul>
<li><code>do</code> 循环要求必须使用关键字 <code>do</code> 来标识循环的开始，用 <code>while</code> 来标识循环的结尾并进入循环条件判断。</li>
<li>和 <code>while</code> 循环不同，<code>do</code> 循环是用分号结尾的。</li>
<li>如果 <code>while</code> 的循环体使用花括号括起来的话，则 <code>while</code> 循环也不用使用分号做结尾。</li>
</ul>
</li>
</ul>
<h2>示例</h2>
<h3>代码示例</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> i = <span class="hljs-number">0</span>;
<span class="hljs-keyword">do</span> {
  i += <span class="hljs-number">2</span>;
} <span class="hljs-keyword">while</span> (i &lt; <span class="hljs-number">10</span>);

<span class="hljs-variable language_">console</span>.<span class="hljs-title function_">log</span>(i); <span class="hljs-comment">// 10</span>
</code></pre>
