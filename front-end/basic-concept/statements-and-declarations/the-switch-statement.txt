c:Tc3c,<h1>switch 语句</h1>
<p><strong><code>switch</code></strong> 语句允许一个程序求一个表达式的值并且尝试去匹配表达式的值到一个 <code>case</code> 标签。如果匹配成功，这个程序执行相关的语句。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">switch</span> (expression) {
   <span class="hljs-keyword">case</span> <span class="hljs-attr">value_1</span>:
      statements_1
      [<span class="hljs-keyword">break</span>;]
   <span class="hljs-keyword">case</span> <span class="hljs-attr">value_2</span>:
      statements_2
      [<span class="hljs-keyword">break</span>;]
   ...
   <span class="hljs-attr">default</span>:
      statements_def
      [<span class="hljs-keyword">break</span>;]
}
</code></pre>
<p>工作原理：首先设置表达式 <code>expression</code>（通常是一个变量）。随后表达式的值会与结构中的每个 <code>case</code> 的值做比较。如果存在匹配，则与该 <code>case</code> 关联的代码块会被执行。请使用 <code>break</code> 来阻止代码自动地向下一个 <code>case</code> 运行。</p>
<h3>参数</h3>
<table>
<thead>
<tr>
<th>参数</th>
<th>说明</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>expression</code></td>
<td>用于比较的表达式</td>
</tr>
<tr>
<td><code>value_(n)</code></td>
<td>与 <code>expression</code> 比较的值</td>
</tr>
<tr>
<td><code>statement(n)</code></td>
<td>执行语句</td>
</tr>
</tbody>
</table>
<h3>关键词</h3>
<ul>
<li><code>case</code>：表示一种情况，如果 <code>expression</code> 等于 <code>value</code> ，就执行 <code>statement</code></li>
<li><code>break</code>：会使代码跳出 <code>switch</code> 语句，如果没有关键词 <code>break</code>，代码执行就会继续进入下一个 <code>case</code> 。</li>
<li><code>default</code>：说明了表达式的结果不等于任何一种情况时的操作（事实上，它相对于 else 从句）。</li>
</ul>

<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> myCar = <span class="hljs-string">&#x27;Porsche&#x27;</span>

    <span class="hljs-keyword">break</span>;
  <span class="hljs-keyword">case</span> <span class="hljs-string">&#x27;Honda&#x27;</span>: <span class="hljs-title function_">alert</span>(<span class="hljs-string">&quot;My car is Honda&quot;</span>);
    <span class="hljs-keyword">break</span>;
  <span class="hljs-keyword">case</span> <span class="hljs-string">&#x27;Porsche&#x27;</span>: <span class="hljs-title function_">alert</span>(<span class="hljs-string">&quot;My car is Porsche&quot;</span>);
    <span class="hljs-keyword">break</span>;
  <span class="hljs-attr">default</span>: <span class="hljs-title function_">alert</span>(<span class="hljs-string">&quot;I have no car&quot;</span>);
}
</code></pre>
