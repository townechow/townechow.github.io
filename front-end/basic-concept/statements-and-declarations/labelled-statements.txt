c:T8db,<h1>label 语句</h1>
<p><strong>标记语句</strong>可以和 <code>break</code> 或 <code>continue</code> 语句一起使用。标记就是在一条语句前面加个可以引用的标识符。</p>
<p><strong>注意</strong>：标记的循环或块非常罕见。通常可以使用函数调用而不是循环跳转。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-attr">identifier</span>: statement;
</code></pre>
<h3>参数</h3>
<table>
<thead>
<tr>
<th>参数</th>
<th>描述</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>identifier</code></td>
<td>任何不是保留关键字的 JavaScript 标识符。</td>
</tr>
<tr>
<td><code>statement</code></td>
<td>语句。<code>break</code> 可用于任何标记语句，而 <code>continue</code> 可用于循环标记语句。</td>
</tr>
</tbody>
</table>
<h3>描述</h3>
<ul>
<li>可使用一个标签来唯一标记一个循环，然后使用 <code>break</code> 或 <code>continue</code> 语句来指示程序是否中断循环或继续执行。</li>
</ul>

<h3>标注示例</h3>
<ul>
<li>用作标签的 <code>identifier</code> 必须是一个合法的 JavaScript 标识符，而不能是一个保留字。</li>
</ul>
