c:T80a,<h1>while 语句</h1>
<p><strong>while 语句</strong>可以在某个条件表达式为真的前提下，循环执行指定的一段代码，直到那个表达式不为 <code>true</code> 时结束循环。</p>
<h2>语法</h2>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">while</span> (expression) statement;
</code></pre>
<h3>参数</h3>
<table>
<thead>
<tr>
<th>参数</th>
<th>描述</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>expression</code></td>
<td>条件表达式，在每次循环前被求值。如果求值为 <code>true</code>，<code>statement</code>就会被执行。如果求值为 <code>false</code>，则跳出 <code>while</code> 循环执行后面的语句。</td>
</tr>
<tr>
<td><code>statement</code></td>
<td>只要条件表达式求值为 <code>true</code>，该语句就会一直被执行。要在循环中执行多条语句，可以使用块语句（<code>{ ... }</code>）包住多条语句。</td>
</tr>
</tbody>
</table>
<p>注意：使用 <code>break</code> 语句在 <code>expression</code> 计算结果为真之前停止循环。</p>
<h2>示例</h2>
<h3>代码示例</h3>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> i = <span class="hljs-number">0</span>;
<span class="hljs-keyword">while</span> (i &lt; <span class="hljs-number">10</span>) {
  i += <span class="hljs-number">2</span>;
}
</code></pre>
<pre class="hljs"><code class="language-js"><span class="hljs-keyword">var</span> cars = [<span class="hljs-string">&#x27;BMW&#x27;</span>, <span class="hljs-string">&#x27;Volvo&#x27;</span>, <span class="hljs-string">&#x27;Saab&#x27;</span>, <span class="hljs-string">&#x27;Ford&#x27;</span>];
<span class="hljs-keyword">var</span> text = <span class="hljs-string">&#x27;&#x27;</span>;
<span class="hljs-keyword">var</span> i = <span class="hljs-number">0</span>;
<span class="hljs-keyword">while</span> (i &lt; cars.<span class="hljs-property">length</span>) {
  text += cars[i] + <span class="hljs-string">&#x27;&lt;br&gt;&#x27;</span>;
  i++;
}
</code></pre>
